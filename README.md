# Web Service REST con cliente en Javascript
 
Gestión de vehículos: crear, leer, actualizar y borrar.

## Proyectos

* [Modelo](https://github.com/imanolhernando/ws_rest_js/tree/master/modelo): Libería Java (jar) para gestionar el modelo de datos.
* [WS REST](https://github.com/imanolhernando/ws_rest_js/tree/master/ws_rest) : Web Service REST( Proveeddor ) para crear, leer, actualizar y borrar un vehiculo.
* [Cliente REST](https://github.com/imanolhernando/ws_rest_js/tree/master/cliente_rest): Web Service REST( Cliente ) para crear, leer, actualizar y borrar un vehiculo en javascript.

## Tecnologías
	Bootstrap 4
	Javascript
	Maven 3.0
	JAVA EE 1.8 
	Spring
	Swagger
	log4j
	MySQL InnoDB
