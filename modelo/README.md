# MODELO
 
Libería Java (jar) para gestionar el modelo de datos.

## Base de datos

* [SQL](https://github.com/imanolhernando/0554/webservice_rest/modelo/sql)
* MySQL InnoDB : Consultas realizadas mediante procedemientos almacenados
 
#### Procedimientos PL/SQL
	pa_coche_baja: Modifica la fecha de baja del vehiculo a la fecha actual
	pa_coche_borrar : Borrado fisico del coche en la base de datos
	pa_coche_getAll: Obtener todos los coches
	pa_coche_getById: Obtener un coche por su identificador
	pa_coche_getByMatricula: Obtener un coche por su matricula
	pa_coche_insertar: Insertar un nuevo coche
	pa_coche_update: Actualiza los campos de un vehiculo
